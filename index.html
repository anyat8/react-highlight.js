react-highlight.js

A lightweight React wrapper around the Highlight.js syntaxt highlighting library

The Highlight component is a lightweight React wrapper around thehighlight.js library. To use it, simply specify a language and provide it with content (children) to highlight. For example:

<Highlight language='css'>
  .example {
    font-weight: bold;
  }
</Highlight>

To use this component simply npm install react-highlight and optionally choose a custom stylesheet fromhighlight.js.

Demo

Try it yourself by specifying a language and content below:

var three = 1 + 2; // This is a comment

Installation

Install this component using NPM like so:

npm install react-highlight.js --save
